# [01sbx](https://zavrzabr.github.io/01sbx-game/index.html) — My first try in making cellular automata
> VERSION: alpha 0.0<br>

**01sbx** is a Turing-complete digital sandbox that allows you to engineer whatever your heart desires. 

Inspired by the computational beauty of **Conway's Game of Life** and **Wireworld**, this project aims to bridge the gap between complex cellular automata and the player. While traditional automata can feel cryptic and overwhelming to ordinary users, **01sbx** redefines the Wireworld concept by introducing intuitive mechanics, structured tutorials, and gameplay elements that unlock the player's full engineering potential...<br>

## Controls & Hotkeys

Toggle between **Camera Mode** and **Construction Mode** using the interface buttons in the top-right corner.

| Action | Control (macOS / Windows) |
| :--- | :--- |
| **Interact / Move Camera** | `Left Mouse Button` |
| **Delete Block** | `Opt + LMB` / `Alt + LMB` |
| **Open Blocks Menu** | `Double-click` on the construction menu button |
| **Pause / Resume** | `Spacebar` |
| **Advance One Tick** | `Tab` |
| **Save World** | `Cmd + S` / `Ctrl + S` *(Choose your custom path and name!)* |
| **Load World** | `Cmd + L` / `Ctrl + L` |

>  **Pro-Tip:** You can dynamically resize the world frame boundary while working inside the **Camera Mode**.<br>
>  **Pro-Tip2:** Don't see small buttons on the up-right of the screen? open the site in fullscreen mode, i will fix this error pretty soon

## Tech Stack & Features
* **Engine:** GameMaker Studio 2 LTS
* **Save System:** High-performance binary buffer serialization (creates super-lightweight `.01map` files).
* **Cross-Platform Support:** Native input handling tailored for both Windows (Ctrl) and macOS (Cmd/Meta) keymaps.
